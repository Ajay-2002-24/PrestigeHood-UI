# ✨ Overview

**PrestigeHood-UI** is the frontend application for PrestigeHood, a smart digital solution designed for apartments, gated communities, and housing societies.

It provides an intuitive dashboard experience for:

- 👤 Residents  
- 🛡️ Security Guards  
- 🏢 Society Admins  

The application integrates with ASP.NET Core backend APIs and real-time SignalR notifications.
